# kyowork

## 프로젝트 소개

**kyowork**는 GitHub Issues를 작업 큐로 삼아 Claude Code가 자동으로 작업을 처리하는 "1인 개발 공장" 저장소입니다. 이슈에 `claude` 라벨을 달거나 `@claude`를 멘션하면 Claude가 컨텍스트를 파악하고, 경로를 스스로 설계해 코드 변경·문서 작성 등을 끝까지 수행한 뒤 브랜치에 커밋하고 PR 링크를 제안합니다. 본업과 개발을 병행하는 환경을 위해 중간보고를 최소화한 비동기 완결형 워크플로를 지향하며, 저장소 공통 규칙과 운영 원칙은 [`CLAUDE.md`](./CLAUDE.md)에 정의되어 있습니다.
